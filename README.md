# Customer Segmentation Dashboard

## Project Overview
This project focuses on analyzing customer behavior using RFM (Recency, Frequency, Monetary) analysis. The goal is to segment customers into meaningful groups and provide actionable business insights.

## Objectives
- Identify high-value customers
- Detect at-risk customers
- Understand customer purchasing patterns
- Support data-driven marketing strategies

## Tools & Technologies
- Power BI
- DAX
- Excel

## Dashboard Features
- KPI Cards (Total Customers, Total Revenue, Avg Spend)
- Customer Segmentation (High Value, Loyal, At Risk)
- Revenue Contribution Analysis (Pie Chart)
- Customer Behavior Analysis (Scatter Plot: Orders vs Spend)
- Interactive Filters

## Key Insights
- A small percentage of customers contribute a large share of revenue
- At-risk customers have low recent activity and require re-engagement
- High-value customers show strong spending patterns
- Customer purchase behavior varies based on frequency and spending

## Conclusion
This project demonstrates how customer segmentation can help businesses improve marketing strategies, retain customers, and increase revenue.
